Программа «Фидла Питон +» – это экспериментальная учебная среда разработки на высокоуровневом языке программирования Python. Он представляет собой редактор кода с подцветкой и возможностью компиляции (рисунок 1):

 

Данная среда разработки обладает следующими функциональными возможностями и свойствами:

  • Открытие файлов Python с расширением PY.
  • Подцветка кода.
  • Компиляция кода Python.
  • Дублирование сообщений об исключениях на русском языке.
  • Сохранение программы.

Приложение работает на компьютере под управлением OC Windows 7 и выше, с установленным NET Framework 2.0+.
